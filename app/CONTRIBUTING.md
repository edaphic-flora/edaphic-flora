# Contributing to Edaphic Flora

Thanks for helping! By contributing, you agree to the following:

## Code
- Code contributions are licensed under **AGPL-3.0-or-later**.
- Include a short description in your PR, and keep changes focused.
- Please add tests or a simple smoke check when possible.

## Data
- Data contributions are licensed under **CC BY-NC 4.0**.
- Only contribute data you have the right to share; include a citation in `references.csv`
  (or in the PR description) when applicable.

## Developer Certificate of Origin (DCO)
By submitting a contribution, you certify:
> I have the right to submit this work under the project's licenses and I am granting
> it under the same licenses indicated above.

## Code of Conduct
This project follows the Contributor Covenant:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/
